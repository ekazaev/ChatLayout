Core

  • CollectionViewChatLayout

    A collection view layout designed to display items in a grid similar to UITableView, while aligning them to the leading or trailing edge of the UICollectionView. This layout facilitates chat-like behavior by maintaining a constant content offset from the bottom. Additionally, it is capable of handling autosizing cells and supplementary views.
